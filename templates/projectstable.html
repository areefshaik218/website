To-Do List

The To-Do List project is a web application developed using HTML, CSS, and Django, designed to help users manage their tasks efficiently. It allows users to create, update, and delete tasks, providing an intuitive interface for organizing daily activities. The application features a clean and responsive design, ensuring usability across devices. Users can categorize tasks based on priority and mark them as completed. With a focus on functionality, this project showcases my ability to implement CRUD operations and work with Django's framework to handle backend logic seamlessly.
